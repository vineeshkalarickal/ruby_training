# IF, ELSE, ELSIF FLOW CONTROL

Type                Example
--------------------------------------------------------------------------------
Conditional         if x == y
execution             puts "First condition is true!"
                    elsif x == z
                      puts "First condition is false!"
                      puts "Second condition is true!"
                    else
                      puts "Both first & second conditions are false!"
                    end

For an exhaustive discussion, consult Ruby 2.5.1 (latest) documentation here:
